# Clean CV Files - External CV Folder

## 📁 Files Available:

### 1. Light Theme CV (Original Design)
- **`index.html`** - Original light theme CV
- **`styles.css`** - Light theme styling
- **To use:** Open `index.html` in browser

### 2. Black Theme A4 CV (Corrected Version)
- **`black-a4.html`** - Black theme, A4 format, CORRECT layout
- **`black-styles.css`** - Black theme styling with proper layout
- **To use:** Open `black-a4.html` in browser

### 3. Shared Resources
- **`profile-photo.jpg`** - Your professional photo
- **`README.md`** - Original documentation

## ✅ Features of Black A4 CV (Correct Version):

### Layout Fixes Applied:
1. **Contact Information on LEFT** - In main content area
2. **No Hebrew** - Only English, Russian, Bulgarian languages
3. **Better Space Usage** - 75%/25% split (main/sidebar)
4. **Compact Design** - Optimized for A4 printing

### Design:
- **Theme:** Black/dark with cyan accents
- **Format:** A4 (210mm × 297mm)
- **Print-ready:** Press Ctrl+P to save as PDF
- **Responsive:** Works on all screen sizes

## 🚀 Quick Start:

### For Black A4 CV:
```bash
# Open the black A4 CV
open black-a4.html      # Mac
xdg-open black-a4.html  # Linux
start black-a4.html     # Windows
```

### For Original Light CV:
```bash
# Open the original light theme
open index.html
```

## 🖨️ To Print/Save as PDF:
1. Open either HTML file
2. Press **Ctrl+P** (or **Cmd+P** on Mac)
3. Choose "Save as PDF"
4. Select A4 paper size
5. Save your CV
