# Exact CV Copy - HTML & CSS Only

This folder contains an exact HTML/CSS copy of Pavel Tarlev's CV website, with no JavaScript, no dynamic features, and no theme switching.

## Files

### 1. `index.html`
- Complete HTML structure matching the original CV
- All content sections: Header, Summary, Tech Stack, Projects, Experience
- Sidebar with Profile, Contact, Languages, Hobbies, Bio
- Uses Font Awesome icons for visual elements
- Semantic HTML structure

### 2. `styles.css`
- Exact styling matching the original CV design
- Orange accent color (#ff6600) for headers and highlights
- Clean, professional layout with cards and shadows
- Responsive design that works on mobile and desktop
- No dark/light theme switching - single light theme only

## Design Features

### Color Scheme
- **Primary Accent:** #ff6600 (orange)
- **Text:** #333 (dark gray)
- **Secondary Text:** #666 (medium gray)
- **Background:** #f8fafc (light gray)
- **Cards:** White with subtle shadows
- **Borders:** #e2e8f0 (light gray)

### Layout
- **Main Content:** 2/3 width, left side
- **Sidebar:** 1/3 width, right side (350px)
- **Responsive:** Stacked on mobile
- **Grid Layout:** CSS Grid for main structure
- **Flexbox:** For internal component layouts

### Typography
- **Font:** System fonts (Apple, BlinkMacSystemFont, Segoe UI, etc.)
- **Headings:** Bold with orange color
- **Body Text:** 16px, 1.6 line height
- **Section Titles:** 24px, orange, with margin bottom

## Content Sections

### Main Content Area
1. **Header** - Name and title
2. **Summary** - Professional overview
3. **Tech Stack** - 20 technology icons with labels
4. **Projects** - 4 project cards with tech tags
5. **Experience** - 3 job experiences with bullet points and tech tags

### Sidebar
1. **Profile** - Name, title, and avatar placeholder
2. **Contact** - Email, phone, LinkedIn, GitHub
3. **Languages** - 4 languages with star ratings
4. **Hobbies & Interests** - 4 hobbies with icons
5. **Bio** - Short personal bio

## How to Use

1. Open `index.html` in any web browser
2. The CV will display exactly as designed
3. No setup or installation required
4. Works offline (except for Font Awesome CDN)

## No JavaScript Features

This version intentionally excludes:
- Theme switching (dark/light mode)
- Interactive collapsible sections
- PDF generation buttons
- Copy-to-clipboard functionality
- Any dynamic JavaScript features

## Browser Compatibility

Works in all modern browsers:
- Chrome, Firefox, Safari, Edge
- Mobile browsers (iOS Safari, Android Chrome)
- Print-friendly (Ctrl+P works well)

## Print/PDF Ready

The design is optimized for printing:
- Clean white background
- Good contrast for text
- Proper page breaks
- Can be printed directly or saved as PDF from browser

## Customization

To update content:
1. Edit text in `index.html`
2. Update styling in `styles.css`
3. Replace placeholder content with actual information
